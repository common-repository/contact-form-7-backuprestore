Backup and restore your Contact Form 7 configuration data

== Description ==

This is a simple WordPress plugin that will allow you to backup and restore
your WordPress Contact Form 7 data / configuration.

It will also enable you can transfer your configuration from one site to
another to save time re-typing those configurations.

Works with: Contact Form 7 version 1.xx

--------------------------

Note: For more information, visit the 
[Contact Form Backup plugin](http://www.codestuff.com/projects/contact-form-backup-plugin/) 
page at  [Code Stuff](http://www.codestuff.com)

== Installation ==

1. Unzip the plugin into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure in Wordpress under Tools / Contact Form Backup.
